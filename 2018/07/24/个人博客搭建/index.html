#个人博客搭建

##前言

这篇文章只是搭建了博客并发布到gitPages上

##准备

###git
安装系统linux自带的git，git版本是1.8.3.1，github支持的git版本是1.7.1以上

环境：CentOS Linux release 7.4.1708
安装命令：yum install git
查看版本：git –version

###hexo
安装hexo之前需安装nodejs hexo官网
安装hexo：

npm install -g hexo-cli

安装hexo部署到git page的deployer：

npm install hexo-deployer-git –save

###gitHub

需要一个github账号，创建一个用来发布hexo文章的仓库
用ssh实现免密码提交`
